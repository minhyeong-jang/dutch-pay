# CalPayment 제품 요구사항 (PM Brief)

> v1.0 MVP 요구사항 정의
> 작성일: 2026-02-23 | 작성자: PM

---

## 1. 타겟 사용자

### Primary: "여행 총무" (28세, 직장인)
- 대학 동기 8명과 분기마다 여행. 매번 총무 역할.
- 결제 8~15건, 각자 번갈아 결제.
- Pain: "토스/카카오페이는 1건만 가능. 여행 전체를 한 번에 못 함."

### Secondary A: "회식 간사" (32세, 팀장)
- 월 1~2회 회식. 1차/2차/3차 참여 인원이 다름.
- Pain: "차수별 인원이 달라서 수기 계산이 귀찮음."

### Secondary B: "동호회 운영자" (26세, 대학원생)
- 월 2회 정기 모임. 참가자 10~20명 유동적.
- Pain: "참가자가 많아서 수기 계산 극도로 번거로움."

## 2. MVP 범위 (v1.0 - 웹 전용)

### MUST HAVE
| # | 기능 | 상세 |
|---|------|------|
| 1 | 모임 CRUD | 생성, 이름 수정, 삭제. localStorage 저장. |
| 2 | 참가자 관리 | 이름 입력, 태그 색상 자동 지정, 삭제. 중복 방지. |
| 3 | 결제 건 CRUD | 사용처, 금액, 결제자, 참여자 다중 선택. |
| 4 | 정산 자동 계산 | `calculateSettlement` + `calculateReceiveSummary` |
| 5 | 정산 결과 표시 | 각자 송금 / 일괄 송금 탭 |
| 6 | 공유 링크 | 읽기 전용 URL. 비로그인 접근 가능. |
| 7 | 텍스트 복사 | 클립보드 복사 (카톡/문자 붙여넣기용) |
| 8 | 카카오톡 공유 | Kakao JS SDK |
| 9 | 반응형 모바일 퍼스트 | Tailwind + shadcn/ui |
| 10 | 랜딩 페이지 | CTA + SEO |

### SHOULD HAVE (v1.1)
- 소셜 로그인 (카카오, 구글)
- 서버 데이터 저장 (Supabase)
- localStorage → DB 마이그레이션
- 정산 완료 체크

### COULD HAVE (v1.2+)
- 네이티브 모바일 앱 (Expo)
- 영수증 OCR, 송금 딥링크, 다국어
- 여행 큐레이션/리뷰 (백로그)

## 3. 핵심 유저 스토리 (12개)

### US-01: 새 모임 생성
**As a** 비로그인 사용자, **I want to** 로그인 없이 바로 모임을 만들 수 있다.
- [x] "바로 시작하기" → 모임 이름 입력 → 생성
- [x] localStorage 저장
- [x] 생성 후 참가자 입력으로 이동

### US-02: 참가자 등록
- Enter 또는 추가 버튼으로 등록
- 태그 색상 자동 할당
- 중복 이름 방지
- 최소 2명

### US-03: 결제 내역 추가
- 사용처, 금액, 결제자, 참여자 입력
- "전체 선택" 기본값
- 실시간 천 단위 콤마
- 1인당 금액 미리보기

### US-04: 결제 내역 삭제
- 확인 다이얼로그 후 삭제
- 정산 결과 자동 재계산

### US-05: 정산 결과 확인
- 각자 송금 / 일괄 송금 탭
- 0원 송금 생략
- 원 단위 내림

### US-06: 공유 링크 생성
- 읽기 전용 URL
- "내 이름 찾기" 기능
- 하단 "나도 정산하기" CTA

### US-07: 카카오톡 공유
- Kakao SDK 공유
- 미설치 시 Web Share API 폴백

### US-08: 텍스트 복사
- 포맷팅된 텍스트 클립보드 복사
- 토스트: "복사되었습니다"

### US-09: 모임 목록 관리
- 카드 형태 목록
- 삭제 확인 다이얼로그
- "이 브라우저에만 저장" 안내

### US-10: 모바일 최적화
- 320~430px 가로 스크롤 없음
- 터치 타겟 44x44px 이상
- `inputmode="numeric"` 숫자 키패드

### US-11: 공유 링크 수신자 경험
- 로그인 없이 즉시 결과 표시
- "내 이름 찾기" 하이라이트
- OG 태그 미리보기

### US-12: 랜딩 페이지
- "토스로 못 하는 복잡한 정산, 1분 만에"
- "바로 시작하기" CTA above the fold
- 기능 요약 3개
